Projects

Here you can see a few projects I've worked on over the past two years as I've developed my skills in software development. WishLister

WishLister was my first project in Ruby on Rails, and was a part of the CodeLabs back-end curriculum. I wanted to build something functional that looked clean and professional. My original goal was to pull product data from major retailers such as Amazon, but I found that they make lots of money charging for access to that data. So I settled for an API that creates fake products with similar data to real e-commerce sites. The site pulls existing products from the API and keeps updated prices and information. User-created products are stored in a database, and can be accessed by all users to the site.

GetFriended

GetFriended, my not-so-original social media hub, began as a part of the curriculum of the Code Labs front-end bootcamp where I began learning and working with Angular. It includes the ability for a user to log in and out securely, add and delete friends from the current roster of users, and add posts to the app's timeline.

  • Check it out

Paducah Coffee Co.

Paducah Coffee Company is my other job, and revamping the website played a role in igniting my passion for software development. The final site is still being developed, but I worked out a functional rough draft while I learned Angular. The site includes e-commerce functionality with a products page and a working cart, as well as a blog element where users logged in as administrators can create, update, and delete blog posts.

  • Check it out

MyRecipes

MyRecipes began as the primary course project of the CodeFi Front-End program where I began working with Angular. I have since modified it to make it my own. It includes the functionality for a user to log in and out securely, to add/update/delete recipes from their page, manage a shopping list that can be updated with the ingredients of a recipe, and store recipe data on a backend framework to be retrieved on a different device or a later date.
