Projects

Here you can see a few projects I've worked on over the past year as I've learned front-end software development. Each image is a link to the GitHub repository containing the project's source code, while each "Check it Out" button will link you directly to the hosted project itself so you can see it in action.


GetFriended

GetFriended, my not-so-original social media hub, began as a part of the curriculum of the CodeLabs front-end bootcamp where I began learning and working with Angular. It includes the ability for a user to log in and out securely, add and delete friends from the current roster of users, and add posts to the app's timeline.

  • Check it out

Paducah Coffee Co.

Paducah Coffee Company is my other job, and revamping the website played a role in igniting my passion for software development. The final site is still being developed, but I worked out a functional rough draft while I learned Angular. The site includes e-commerce functionality with a products page and a working cart, as well as a blog element where users logged in as administrators can create, update, and delete blog posts.

  • Check it out

MyRecipes

MyRecipes began as the primary course project of the CodeFi Front-End program where I began working with Angular. I have since modified it to make it my own. It includes the functionality for a user to log in and out securely, to add/update/delete recipes from their page, manage a shopping list that can be updated with the ingredients of a recipe, and store recipe data on a backend framework to be retrieved on a different device or a later date.

  • Check it out

Etch-A-Sketch

This little Etch-A-Sketch page was built as part of the curriculum of The Odin Project, a wonderful resource for self-teaching web development. It features a user-customizable grid of squares, which will change randomly within a selected spectrum of colors when hovered over by the mouse.

  • Check it out
